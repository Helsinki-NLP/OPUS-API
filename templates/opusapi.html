OPUS-API

Use parameters corpus, source, target, preprocessing and version to search for corpora. Set version to latest to get the latest version. For example:

http://opus.nlpl.eu/opusapi/?corpus=OpenSubtitles&source=en&target=fi&preprocessing=xml&version=latest

List available copora:

http://opus.nlpl.eu/opusapi/?corpora=True

List available languages:

http://opus.nlpl.eu/opusapi/?languages=True

List available languages for a corpus:

http://opus.nlpl.eu/opusapi/?languages=True&corpus=OpenSubtitles

List available target languages for a source language:

http://opus.nlpl.eu/opusapi/?languages=True&source=fi

List available target languages for a source language in a corpus:

http://opus.nlpl.eu/opusapi/?languages=True&corpus=OpenSubtitles&source=fi
